# Resumo do Lab: Localizando Serviços no Azure

## Introdução
Este repositório contém o resumo do aprendizado obtido durante o laboratório **"Localizando Serviços por Categoria no Azure"**, parte do curso preparatório para a certificação **AZ-900** na [DIO](https://www.dio.me/).

O objetivo principal foi navegar pelo Portal do Azure e compreender a taxonomia e organização dos serviços em nuvem oferecidos pela Microsoft.

## Principais Aprendizados

O Portal do Azure é o console unificado onde gerenciamos todos os recursos. Durante o laboratório, explorei o menu **"Todos os Serviços"** e identifiquei como eles são agrupados para facilitar a administração:

### 1. Categorias de Serviços
A Microsoft agrupa centenas de serviços em categorias lógicas. As principais que explorei foram:

* **Computação (Compute):** Engloba serviços de processamento como **Máquinas Virtuais (VMs)**, **App Services** e **Azure Functions** (Serverless).
* **Rede (Networking):** Fundamental para conectividade, incluindo **Virtual Networks (VNet)**, **Load Balancers** e **VPN Gateways**.
* **Armazenamento (Storage):** Soluções para guardar dados, como **Azure Blob Storage**, **Arquivos** e **Discos Gerenciados**.
* **Identidade (Identity):** Onde fica o **Microsoft Entra ID** (antigo Azure AD), crucial para autenticação e controle de acesso.

### 2. Marketplace
Além dos serviços nativos, o Azure Marketplace oferece soluções de terceiros (como Firewalls da Fortinet ou Imagens Linux específicas) que podem ser integradas ao ambiente Azure com poucos cliques.

### 3. Ferramentas de Busca
Aprendi que a barra de pesquisa global é a ferramenta mais ágil para um administrador. Ela permite localizar rapidamente qualquer recurso, serviço ou documentação sem precisar navegar por todos os menus.

## Conclusão
A organização por categorias no Azure é essencial para a governança de TI. Para um profissional de infraestrutura, entender onde cada recurso se encaixa (se é IaaS, PaaS ou SaaS) e em qual categoria ele reside é o primeiro passo para desenhar arquiteturas robustas e seguras na nuvem.
